Running bash script for compilation

Syntax

./compile_w_lib.sh C_SOURCEFILE.c [-lib] # With library recompilation
./compile_w_lib.sh C_SOURCEFILE.c # Only main file recompilation

-lib flag provides control whether to recompile shared library boyz_RP_library

This bash script then produces executable binary TCP_server

Required files

- Main c. file must be located in `/root`   
- library header and source files must be in `/root/include`   
- Script file `compile_w_lib.sh` must be in `/root`

Problem solving

Settings permission for script:

chmod +x compile_w_lib.sh
