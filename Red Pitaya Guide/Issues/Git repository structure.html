Git repository structure

  • arm - TCP server C code + compile bash scripts
    • include - C library with functions for Red Pitaya
  • ddr_patch - Device tree and U-boot files for reserving a portion of onboard DDR memory
  • doc - Register maps of IP cores
  • gui - Matlab App Designer GUI source files
    • sources - utility functions used primarily for GUI (some duplicity can exist with the other utility folder)
  • impl - Sources for everything Programmable-logic related
    • bds - Vivado block designs (used mainly for rebuilding the reference project)
    • bitfile - Used to store the "stable release" of particular bitstream
    • matlab - System generator IP cores source files
    • proj - Vivado projects
    • srcs
      • hdl - "raw" HDL modules
      • ips - custom IP cores, mostly generated by System Generator
    • tcls - TLC scripts for Vivado
    • xdcs - hardware constraints files for Vivado
  • sim - Development, theoretical and proof-of-concept oriented files
  • utility - Folder for various Matlab functions used mainly in sim folder
